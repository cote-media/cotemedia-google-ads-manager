# Three buckets for every git-tracked .md/.txt:
#   GATED   — load-bearing, read EVERY session → in SOURCE_DOCS (scripts/build-resume-digest.mjs), freshness-checked.

# of them before commit. Promotion to GATED is a deliberate Russ-level call (never automatic). When in doubt → TRACKED
# (hash-recorded, never lost), NOT IGNORE.
#
